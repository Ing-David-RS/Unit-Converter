# Unit-Converter-script

## Length_Converter
In this case, the conversion only works for:

   - **Meter (m)**
   - **Centimeter (cm)**
   - **Foot (ft)**
   - **Inch (in)**
## How is this works
Into terminal of linux put the following:

_./LC_script_ <parameter> <magnitude_of_unit>

###### The parameters are the followings:
- m2cm
- m2ft
- m2in
- cm2m
- cm2ft
- cm2in
- ft2m
- ft2cm
- ft2in
- in2m
- in2cm
- in2ft
